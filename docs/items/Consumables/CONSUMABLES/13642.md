# Start your Journey Pack (ID: 13642)

### Description
    The Start Your Journey Pack contains several items that will allow you to move around the cities of the Rune-Midgard kingdom so that you can quickly complete your class change quest, gain more EXP for 30 minutes, and stay healed during the initial battles of the adventure.
    20 Red Envelopes are also provided to help you get that last bit of experience needed for the next Level.
    Contains:
    20 Red Envelopes
    200 Apprentice Red Potions
    1 Battle Manual
    3 Rune-Midgard Butterfly Wings
    Can only be stored.
