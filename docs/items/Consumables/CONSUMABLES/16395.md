# HD Elunium Box(5) (ID: 16395)

### Description
    A box containing 5 DE Eluniums inside.
    Item Description:
    High Density Elunium that was buried deeply.
    Used as material to strengthen armor currently from Refinement Levels 7 up to 9.
    The density of this item guarantees that Refine attempts will not break the armor, and if the attempt fails, it will only be downgraded by 1 level.
    Only a Blacksmith named Mighty Hammer can use this ore to Refine.
    This item is non-refundable.
    Weight: 1
