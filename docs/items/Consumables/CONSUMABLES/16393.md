# HD Oridecon Box(5) (ID: 16393)

### Description
    A box that contains 5 Oridecons inside.
    Item Description:
    High Density Oridecon that was buried deep.
    Used as material to strengthen weapons currently from Refinement Levels 7 up to 9.
    The density of this item ensures that Refine attempts will not break the weapon, and if the attempt fails, it will only be downgraded by 1 level.
    Only a Blacksmith named Mighty Hammer can use this ore to Refine.
    This item is non-refundable.
    Weight: 1
