# Heart Pact (ID: 6413)

### Description
    A heart contract that nullifies the Experience penalty upon being knocked out (KO).
    If you have this item in your inventory when you are knocked out (KO), it will be used to prevent experience loss.
    Even if the current experience is 0%, this item will still be used.
