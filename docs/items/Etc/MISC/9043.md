# Marionette Egg (ID: 9043)

### Description
    An egg in which a Marionette pet rests.
    Can be hatched using a Pet Incubator.
    When intimacy is Cordial:
    when the user receives physical attacks, adds a chance to increase Neutral resistance by 20% for 3 seconds and recover 100 HP.
    When intimacy is Loyal:
    when the user receives physical attacks, adds a chance to increase Neutral resistance by 20% for 5 seconds and recover 100 HP.
    Type: Monster Egg
    Diet: Small Snow Flower
