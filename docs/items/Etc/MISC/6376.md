# KVM Badge (ID: 6376)

### Description
    An emblem provided to warriors who brought honor to the Krieger Von Midgard field.
    This emblem can be used as KVM Points and exchanged for rewards with the KVM Logistics Officer in the Battlefield Waiting Room.
    Personal KVM Points will be used before this item is used for the reward requirement.
    Speak to a Maroll Battle Recruiter in Prontera, Morocc, Aldebaran, Geffen, Payon, Lighthalzen, or Rachel to be sent to the Battlefield Room.
