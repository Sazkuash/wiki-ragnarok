# Imp Egg (ID: 9056)

### Description
    An egg in which an Imp pet rests.
    Can be hatched using a Pet Incubator.
    When intimacy is Cordial:
    Decreases damage received from Fire element attacks by 2%.
    Increases Physical Damage against Fire element enemies by 1%.
    When intimacy is Loyal:
    Decreases damage received from Fire element attacks by 3%.
    Increases Physical Damage against enemies of element Fire by 2%.
    Type: Monster Egg
    Diet: Flame Gem
