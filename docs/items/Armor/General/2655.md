# Bloody Iron Ball (ID: 2655)

## Description

> However, prisoners equipped with this ball managed to use it as a weapon.  
> When equipped with Shackle:

> Decreases damage received from Zombie Prisoner and Skeleton Prisoner by 20%.  
> Type: Accessory  
> Weight: 400  
> Armor Level: 1  
> Requirement:
