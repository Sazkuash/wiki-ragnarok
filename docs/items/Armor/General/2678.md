# Ring of Flame Lord (ID: 2678)

## Description
> A ring imbued with the power of the fire spirit king, which can be used to strengthen the user  
> This ring forms a corresponding set with the Ring of Resonance.  
> VIT +1  
> STR +2  
> ATK +15  
> Reduces damage received from Fire property attacks by 10%.  
> Random chance to automatically cast various skills when dealing physical damage.  
> When equipped with Ring of Resonance, this ring adds the Auto-Cast skill.  
> Type: Accessory  
> Weight: 10  
> Armor Level: 1  
> Requirement:  
> 2nd Transcendent Classes
